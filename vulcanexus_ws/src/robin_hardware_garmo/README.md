# Garmo Laser Profilometer Driver (Example Domain Adapter)

> **Note:** This is an **example domain adapter**, not a reusable ROBIN module.
> It shows how to integrate a laser profilometer sensor with the ROBIN telemetry
> pipeline (Module 3). Use this as a reference when writing your own sensor adapter.

This package provides a ROS 2 driver for the Garmo laser profilometer, which
measures bead geometry (height/width) in the welding reference profile.
